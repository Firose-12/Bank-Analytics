# Bank-Analytics

# Project Overview

This project focuses on analyzing banking data to understand customer behavior, financial performance, loan trends, and business insights through interactive dashboards and data visualization techniques.

# Tools & Technologies Used

* Excel
* Power BI
* Tableau
* SQL

# Key KPIs

* Total Customers
* Loan Distribution
* Account Balance Analysis
* Customer Segmentation
* Revenue & Financial Performance
* Branch Performance Analysis

# Features

* Interactive Dashboard
* Data Cleaning & Transformation
* KPI Tracking
* Trend Analysis
* Customer Insights
* Regional & Branch Analysis
* Slicers and Filters

# Project Insights

* Identified customer banking patterns
* Analyzed loan and account distribution
* Compared branch-wise performance
* Tracked financial growth trends
* Generated business insights for decision-making

# Files Included

* Dataset Files
* Dashboard Files
* Project Report
* Dashboard Screenshots

# Conclusion

This project demonstrates practical banking data analytics skills by converting raw financial data into meaningful insights using visualization and business intelligence tools.
